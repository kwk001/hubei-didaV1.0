# 前端设计
## 简介
- 打造独具特色、设计品质卓越的生产级前端界面。当用户要求构建网页组件、页面、作品、海报或应用程序时（示例包括网站、落地页、仪表板、React组件、HTML/CSS布局），或当需要对任何网页用户界面（UI）进行样式设计/美化时，请运用此技能。生成富有创意、精雕细琢的代码和UI设计，避免千篇一律的人工智能美学。
- 这项技能指导创建独特且达到生产级的前端界面，避免出现普通的“AI杂乱”美学。在实现真正可运行的代码时，要格外关注美学细节和创意选择。

## 设计思维
1. **在编码之前，先了解上下文，并确定一个大胆（BOLD）的美学方向：**
- 目的：这个界面解决了什么问题？谁在使用它？
- 风格：选择一种极端风格：极简主义、极繁主义、复古未来主义、有机/自然风格、奢华/精致风格、趣味/玩具风格、编辑/杂志风格、野兽派/原始风格、装饰艺术/几何风格、柔和/粉彩风格、工业/实用风格等。有如此多的风格可供选择。利用这些风格作为灵感，但设计出一种忠于美学方向的作品。
- 约束条件：技术要求（框架、性能、可访问性）。
- 差异化：是什么让这个活动令人难忘？有什么特别之处会让人铭记在心？

2. **关键要点：选择一个清晰的概念方向，并精准执行。大胆的极繁主义和精致的极简主义都可行——关键在于意图，而非强度。**

3. **然后实现可运行的代码（HTML/CSS/JS、React、Vue等），该代码应：**
- 生产级且功能齐全
- 视觉上引人注目且令人难忘
- 具有清晰审美观点的凝聚力
- 每个细节都经过精心打磨

## 前端美学指南
1. **关注：**
- 字体设计：选择美观、独特且有趣的字体。避免使用Arial和Inter等通用字体；而应选择能提升前端美观度的独特字体；选择出人意料的、有特色的字体。将独特的显示字体与精致的正文字体搭配使用。
- 颜色与主题：致力于打造统一的美学风格。使用CSS变量确保一致性。主色调搭配鲜明点缀，胜过胆小怯懦、色彩分布均匀的调色板。
- 动态效果：使用动画来增强效果和微交互。对于HTML，优先考虑仅使用CSS的解决方案。在可用时，使用React的Motion库。关注高影响力的时刻：一个精心编排的页面加载，通过交错展示（动画延迟）比分散的微交互更能带来愉悦感。使用滚动触发和悬停状态来制造惊喜。
- 空间构成：意想不到的布局。不对称性。重叠。对角线流动。打破网格的元素。宽敞的负空间或受控的密度。
- 背景与视觉细节：营造氛围和深度，而非仅使用纯色。添加与整体美学相匹配的情境效果和纹理。应用创意形式，如渐变网格、噪点纹理、几何图案、分层透明度、戏剧性阴影、装饰性边框、自定义光标和颗粒叠加。

2. **切勿使用通用的人工智能生成的美学元素，如过度使用的字体系列（如Inter、Roboto、Arial、系统字体）、陈旧的配色方案（尤其是白色背景上的紫色渐变）、可预测的布局和组件模式，以及缺乏特定情境特征的千篇一律的设计。**

3. **进行创造性解读，并做出让人感觉真正为该情境量身打造的出人意料的选择。任何设计都不应千篇一律。应采用明暗不同的主题、不同的字体、不同的美学风格。各代设计绝不能趋同于共同的选择（例如Space Grotesk字体）。**

4. **重要提示：使实现复杂度与美学愿景相匹配。极繁主义设计需要复杂的代码，并融入丰富的动画和效果。极简主义或精致设计则需要克制、精确，以及对间距、排版和细微细节的细致关注。优雅源于对愿景的出色执行。**

5. **记住：克劳德有能力创作出非凡的作品。不要拘束，展现当跳出固有思维模式并全身心投入独特愿景时，真正能够创造出的东西。**

## 前端页面制作规范
1. iCON
- 导航菜单栏里 各页面菜单的iCON，结合对应业务，使用“扁平化+面性”样式的iCON（可以参考https://www.iconfont.cn）
2. 功能点交互
- 页面上的各功能点，点击后必须得有交互，和对应的交互页面！
3. “表格型”页面
- 页面表格，宽度、内容自适应
  - 列固定：第1列、操作栏必须固定
  - 表格内容若超过表格宽度，必须支持自适应固定区域内左右滑动的交互方式查看
- 页面表格，操作栏宽度自适应
  - 操作栏的功能点若小于或等于3个时，自适应宽度
  - 操作栏的功能点若大于3个时，用下拉箭头点击展开查看
- 操作栏的功能点，必须使用文字型！
- 表格第1列必须是序号
- 页面表格里的多选框显隐
  - 若有增删改功能，在表格的第1列，必须有多选框
  - 若没有增删改功能，表格里不能有多选框
- 鼠标移至对应行数据时，该行必须得有相应统一的交互效果
- 状态显示：各类状态必须用颜色标识
- 分页显示
  - 必须展示数据总量
  - 默认一页10行，支持点击切换成1页50行、100行
  - 必须支持点击上、下页切换
  - 支持输入页面数后，直接定位到对应页数据
4. 树状图：需要支持点击收起、展开；收起、展开后页面内容需要自动自适应适配
