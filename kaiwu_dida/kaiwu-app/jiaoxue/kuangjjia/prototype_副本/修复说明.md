# 教师端页面修复脚本使用说明

## 脚本功能
为教师端表格页面添加以下功能：
1. **分页组件增强**：每页行数切换（10/50/100）、页码输入跳转
2. **表格多选框**：全选功能和行选择框

## 使用方法

### 1. 修复单个文件
```bash
cd "/Users/apple/Desktop/文件/kaiwu_dida/kaiwu-app/教学管理/框架/prototype"
python3 fix_pagination_checkbox.py 教师端_辅导教材.html
```

### 2. 批量修复多个文件
```bash
cd "/Users/apple/Desktop/文件/kaiwu_dida/kaiwu-app/教学管理/框架/prototype"
for file in 教师端_辅导教材.html 教师端_试卷管理.html 教师端_实训题库_列表.html; do
    python3 fix_pagination_checkbox.py "$file"
done
```

### 3. 修复所有表格页面
```bash
cd "/Users/apple/Desktop/文件/kaiwu_dida/kaiwu-app/教学管理/框架/prototype"
for file in 教师端_*.html; do
    if grep -q "pagination-info.*共.*条记录" "$file"; then
        python3 fix_pagination_checkbox.py "$file"
    fi
done
```

## 修复内容详情

### 分页组件增强
修复前：
```html
<div class="pagination">
    <span class="pagination-info">共 5 条记录</span>
    <button class="pagination-btn" disabled>上一页</button>
    <button class="pagination-btn active">1</button>
    <button class="pagination-btn">下一页</button>
</div>
```

修复后：
```html
<div class="pagination">
    <span class="pagination-info">共 <span id="myVideoTotal">5</span> 条记录</span>
    <div class="pagination-size">
        <span>每页</span>
        <select id="myVideoPageSize" onchange="changePageSize('myVideo')">
            <option value="10" selected>10</option>
            <option value="50">50</option>
            <option value="100">100</option>
        </select>
        <span>条</span>
    </div>
    <button class="pagination-btn" disabled>上一页</button>
    <button class="pagination-btn active">1</button>
    <button class="pagination-btn">下一页</button>
    <div class="pagination-jump">
        <span>前往</span>
        <input type="number" id="myVideoJumpPage" min="1" value="1">
        <span>页</span>
    </div>
</div>
```

### 表格多选框
修复前：
```html
<thead>
    <tr>
        <th style="width: 60px;">序号</th>
        ...
    </tr>
</thead>
```

修复后：
```html
<thead>
    <tr>
        <th class="checkbox-cell"><input type="checkbox" id="selectAllMyVideo" onclick="toggleSelectAll('myVideo')"></th>
        <th style="width: 60px;">序号</th>
        ...
    </tr>
</thead>
```

## 注意事项

1. **脚本限制**：当前脚本主要修复分页样式和JavaScript函数，表格HTML结构的多选框需要手动添加
2. **备份建议**：运行脚本前建议备份原文件
3. **验证修复**：修复后请在浏览器中验证功能是否正常

## 手动修复步骤（如果脚本不完整）

如果脚本没有完全修复，可以参照 `教师端_视频教材.html` 的修改内容，手动复制以下部分：

1. **CSS样式**：复制 `.pagination-size`、`.pagination-jump`、`.checkbox-cell` 样式
2. **JavaScript函数**：复制 `toggleSelectAll` 和 `changePageSize` 函数
3. **HTML结构**：在表格头部添加多选框列，在分页组件中添加每页行数选择和页码输入
