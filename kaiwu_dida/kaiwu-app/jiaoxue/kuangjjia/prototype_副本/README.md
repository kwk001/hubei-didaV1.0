# 教学实训平台 - 原型系统

## 项目简介

教学实训平台前端原型系统，包含教师端、学生端、教务管理端等多个模块。

## 访问地址

- Gitee Pages: https://kwk001.gitee.io/kaiwu-dida

## 主要功能模块

### 教师端
- 实训任务管理
- 试卷管理
- 题库管理
- 学生管理
- 刀具管理
- 项目管理

### 学生端
- 我的课程
- 我的考试
- 我的实训
- 错题集

### 教务管理端
- 课程管理
- 班级管理
- 教师管理
- 学生管理

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)
- Tailwind CSS (CDN)
- Font Awesome (CDN)

## 本地预览

```bash
# 进入项目目录
cd prototype

# 启动本地服务器
python3 -m http.server 8092

# 浏览器访问
http://localhost:8092
```

## 部署说明

本项目为纯静态网站，可直接部署到任意静态托管服务：
- Gitee Pages
- GitHub Pages
- Vercel
- Netlify
