## 三、响应式布局

### 3.1 问题: 页面未自适应，部分内容被挡住

**解决方案**:
```css
.content-body {
    flex: 1;
    overflow-y: auto;
    padding: 24px;
}

.form-container {
    max-width: 100%;
    overflow-x: hidden;
}

.main-content {
    flex: 1;
    display: flex;
    flex-direction: column;
    overflow: hidden;
}
```

### 3.2 关键 CSS 属性
- `flex-wrap: wrap` - 允许元素换行
- `min-width` - 设置最小宽度防止过度压缩
- `overflow` - 控制溢出行为

## 五、遇到的问题与解决方案

### 5.1 导航加载失败
- **问题**: 404 错误
- **解决**: 使用正确的服务器端口和路径

### 5.2 筛选条件收起时内容未隐藏
- **问题**: 红框区域内容仍然显示
- **解决**: 添加 `.filter-item-extra` 类并设置 `display: none`

### 5.3 分页样式不一致
- **问题**: 分页组件与教师管理页不同
- **解决**: 统一 CSS、HTML 结构和 JavaScript 函数

### 5.4 操作栏宽度固定
- **问题**: 宽度固定为 150px
- **解决**: 改为 `width: auto` 和 `white-space: nowrap`

### 5.5 页面响应式问题
- **问题**: 内容被右侧遮挡
- **解决**: 调整 `.content-body`、`.form-container` 等元素的 overflow 和 width 属性

### 5.6 用户菜单 hover 不生效
- **问题**: 下拉菜单不显示
- **解决**: 
  1. 添加 `!important` 提高 CSS 优先级
  2. 添加 `.user-menu.active` 备选方案
  3. 确保 `.user-menu` 有 `position: relative`

### 5.7 个人中心弹窗被遮挡
- **问题**: 弹窗显示在内容下方
- **解决**: 将弹窗移动到 `body` 直接子元素位置

## 六、最佳实践

### 6.1 CSS 命名规范
- 使用 BEM 命名法: `.block__element--modifier`
- 语义化类名: `.user-menu` 而非 `.um`

### 6.2 JavaScript 代码组织
- 功能模块化
- 事件委托减少监听器数量
- 使用 `DOMContentLoaded` 确保 DOM 加载完成

### 6.3 响应式设计原则
- 移动优先或桌面优先要明确
- 使用相对单位 (rem, %, vw/vh)
- 测试多种屏幕尺寸

### 6.4 浏览器兼容性
- 使用 CSS 前缀工具 (Autoprefixer)
- 测试主流浏览器 (Chrome, Firefox, Safari, Edge)
