## 五、交互效果规范

### 5.1 悬停效果

| 元素 | 悬停效果 |
|------|----------|
| 统计卡片 | `transform: translateY(-4px)` + 阴影增强 |
| 主按钮 | `transform: translateY(-2px)` + 阴影增强 |
| 默认按钮 | 边框变主色 + 背景变浅紫 + 上浮 `1px` |
| 表格行 | 背景变为 `rgba(99, 102, 241, 0.03)` |

### 5.2 过渡动画

| 属性 | 值 |
|------|-----|
| 默认过渡 | `all 0.3s ease` |
| 按钮过渡 | `all 0.3s` |
| 图标过渡 | `all 0.4s cubic-bezier(0.34, 1.56, 0.64, 1)` |

## 六、响应式规范

### 6.1 断点

| 断点 | 宽度 |
|------|------|
| 桌面端 | > 1024px |
| 平板端 | 768px - 1024px |
| 移动端 | < 768px |

### 6.2 移动端适配

- 统计卡片变为 `2列` 布局
- 筛选项变为 `100%` 宽度
- 表格支持横向滚动
- 侧边栏收起

## 八、使用示例

### 8.1 统计卡片

```html
<div class="stats-grid">
    <div class="stat-card">
        <div class="stat-icon">
            <svg>...</svg>
        </div>
        <div class="stat-info">
            <div class="stat-value">128</div>
            <div class="stat-label">教师总数</div>
        </div>
    </div>
</div>
```

### 8.2 筛选区域

```html
<div class="filter-section">
    <div class="filter-row">
        <div class="filter-item">
            <label>姓名</label>
            <input type="text" placeholder="请输入姓名">
        </div>
        <div class="filter-actions">
            <button class="btn-search">查询</button>
            <button class="btn-reset">重置</button>
        </div>
    </div>
</div>
```

### 8.3 表格容器

```html
<div class="table-container">
    <div class="action-bar">
        <div class="action-left">
            <button class="btn-primary">新增</button>
            <button class="btn-default">导出</button>
        </div>
    </div>
    <table>...</table>
    <div class="pagination">...</div>
</div>
```
