# 算法艺术风格

本文档列出了可以使用算法艺术 skill 创建的各种艺术风格类型。每种风格代表一种独特的计算美学运动,可以通过代码表达。

## 基于核心算法哲学的风格

### 1. 有机湍流
- **哲学**: 受自然法则约束的混沌,从无序中产生秩序
- **计算表达**: 由分层 Perlin 噪声驱动的流场,成千上万的粒子沿着矢量力运动
- **视觉特征**: 有机密度图,湍流区域和平静区域,基于速度的着色

### 2. 量子谐波
- **哲学**: 离散实体表现出波状干涉模式
- **计算表达**: 具有相位值的粒子通过正弦波演化,产生干涉模式
- **视觉特征**: 建设性/破坏性干涉产生明亮的节点和空隙,涌现的曼陀罗

### 3. 递归低语
- **哲学**: 跨尺度的自相似性,有限空间中的无限深度
- **计算表达**: 具有递归细分的分支结构和 L 系统
- **视觉特征**: 数学树状形式,带有噪声的对称性破缺,递减的线条粗细

### 4. 场动力学
- **哲学**: 通过对物质的影响使不可见的力可见
- **计算表达**: 来自数学函数的矢量场,沿场线的粒子流
- **视觉特征**: 显示力相互作用的轨迹,不可见力的幽灵般证据

### 5. 随机结晶
- **哲学**: 随机过程结晶成有序结构
- **计算表达**: 圆填充或 Voronoi 镶嵌与松弛算法
- **视觉特征**: 有机铺砖,基于平衡的单元结构,基于大小/接近度的着色

## 其他计算美学运动

### 6. 涌现的静止
- **哲学**: 运动冻结成完美平衡的时刻
- **计算表达**: 达到平衡状态的粒子系统,基于力的定位
- **视觉特征**: 平衡的构图,最小的运动,和谐的排列

### 7. 混沌吸引子
- **哲学**: 具有有界不可预测性的确定性混沌
- **计算表达**: 数学吸引子方程(Lorenz、Rössler),迭代系统
- **视觉特征**: 复杂的轨迹,有界区域,对初始条件的敏感依赖

### 8. 分形景观
- **哲学**: 从简单的递归规则中产生的无限复杂性
- **计算表达**: 使用分数布朗运动生成高度图,菱形-正方形算法
- **视觉特征**: 山脉,海岸线,具有自相似特征的逼真地形

### 9. 元胞自动机花园
- **哲学**: 从简单的基于规则的交互中产生生命
- **计算表达**: Conway 生命游戏变体,连续自动机,多状态系统
- **视觉特征**: 演化模式,涌现行为,有机生长模拟

### 10. 神经美学
- **哲学**: 人工神经网络作为艺术表达
- **计算表达**: 具有激活模式的模拟神经网络,连接权重可视化
- **视觉特征**: 网络图,激活热图,连接强度可视化

### 11. 波干涉场
- **哲学**: 重叠波创建复杂的干涉模式
- **计算表达**: 具有叠加的多个波源,频率调制
- **视觉特征**: 莫尔图案,驻波,共振区

### 12. 群体智能
- **哲学**: 从简单的个体代理中产生的集体行为
- **计算表达**: Boids 算法,蚁群优化,粒子群
- **视觉特征**: 群集模式,涌现的编队,集体决策可视化

### 13. 数学交响曲
- **哲学**: 纯数学关系创造视觉和谐
- **计算表达**: 几何构造,黄金比例应用,三角模式
- **视觉特征**: 完美的对称性,比例之美,数学精度

### 14. 噪声地形
- **哲学**: 随机性雕刻成结构化景观
- **计算表达**: 多倍频噪声函数,域扭曲,湍流效应
- **视觉特征**: 云状编队,地形特征,有机纹理变化

### 15. 时间回声
- **哲学**: 时间作为静态构图中的可见维度
- **计算表达**: 基于时间的粒子轨迹,衰减函数,历史状态可视化
- **视觉特征**: 运动轨迹,褪色的记忆,时间分层效果

## 实现注意事项

每种风格都可以使用 p5.js 框架实现,包括:

1. **种子随机性**: 使用 `randomSeed()` 和 `noiseSeed()` 确保可重现的结果
2. **交互参数**: 用户可控制的影响算法行为的变量
3. **实时探索**: 参数更改时的即时视觉反馈
4. **高度工艺性**: 精心调整的算法,看起来像专业开发的

关键是创建一个指导实现的算法哲学,而不是简单地应用视觉效果。每件作品都应该感觉像是深度计算专业知识和精心优化的产物。
