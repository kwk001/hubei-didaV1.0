# Theme Factory 技能

此技能提供精心策划的专业字体和颜色主题集合,每个主题都配有精心挑选的调色板和字体组合。选择主题后,可以将其应用于任何工件。

## 目的

要对演示幻灯片应用一致、专业的样式,可使用此技能。每个主题包括:
- 具有十六进制代码的协调配色方案
- 用于标题和正文的互补字体组合
- 适合不同环境和受众的独特视觉标识

## 使用说明

要对幻灯片或其他工件应用样式:

1. **展示主题展示**: 显示 `theme-showcase.pdf` 文件,让用户直观地看到所有可用主题。不要对其进行任何修改;只需显示文件供查看。
2. **询问他们的选择**: 询问要应用于展示的主题
3. **等待选择**: 获得关于所选主题的明确确认
4. **应用主题**: 选择主题后,将所选主题的颜色和字体应用于展示/工件

## 可用主题

以下 10 个主题可用,每个主题都在 `theme-showcase.pdf` 中展示:

1. **Ocean Depths** - 专业而平静的海洋主题
2. **Sunset Boulevard** - 温暖而充满活力的日落色彩
3. **Forest Canopy** - 自然而扎实的大地色调
4. **Modern Minimalist** - 干净而现代的灰度
5. **Golden Hour** - 丰富而温暖的秋季调色板
6. **Arctic Frost** - 凉爽而清新的冬季风格主题
7. **Desert Rose** - 柔和而精致的灰褐色调
8. **Tech Innovation** - 大胆而现代的科技美学
9. **Botanical Garden** - 清新而有机的花园色彩
10. **Midnight Galaxy** - 戏剧性的宇宙深色调

## 主题详情

每个主题都在 `themes/` 目录中定义,包含完整规范:
- 具有十六进制代码的协调配色方案
- 用于标题和正文的互补字体组合
- 适合不同环境和受众的独特视觉标识

## 应用流程

选择首选主题后:
1. 从 `themes/` 目录读取相应的主题文件
2. 在整个展示中一致地应用指定的颜色和字体
3. 确保适当的对比度和可读性
4. 在所有幻灯片中保持主题的视觉标识

## 创建自己的主题
要处理现有主题都不适用于工件的情况,可创建自定义主题。根据提供的输入,生成类似上述的新主题。为主题命名时使用类似的描述性名称来描述字体/颜色组合所代表的内容。使用提供的任何基本描述来选择适当的颜色/字体。生成主题后,展示它以供审查和验证。随后,按上述方式应用主题。
